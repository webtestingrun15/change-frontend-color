Plugin that changes the frontend background color of a website

== Description ==

Tweaked from the guides of https://code.tutsplus.com/tutorials/creating-custom-admin-pages-in-wordpress-1--cms-26829 and https://wpbeaches.com/add-an-rgba-color-picker-to-a-wordpress-plugin-input-field/.


== Installation ==

1. Upload `change-frontend-color folder` to the `/wp-content/plugins/` directory
2. Activate the plugin through the 'Plugins' menu in WordPress
3. Go to Tool Settings > Change Frontend Color Settings > Pick A Color > Save Settings
4. Go to Frontend
